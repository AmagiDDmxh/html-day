在 2025年8月2日，星期六，下午4点到6点半，我们将在广州聚会，参与 HTML Day。

什么是 HTML Day？是由 HTML Energy 发起的每年一次写 HTML 的聚会。

目标

  • 让人知道 Web 这个古老的技术还存在，让人体会与感受手工制作网页的乐趣。
  • 还记得你第一次写 HTML 的感觉吗？如果没有学过也没关系哦，一起来玩。
  • 感受 HTML 的（Energy），手写 HTML，就跟摸摸草地一样！
  • 希望每个人都能掌握一些基础技术，自己表达、创作，而不是只依靠大公司决定我们在网上看到什么。

参加活动需要什么

  • 你自己本人
  • 一个可以写 HTML 的东西（电脑、笔记）
  • 可选：铺草地用的毛毯

参加地点

一处带草地的公园，如果下雨就去附近的咖啡店、屋里，进群跟进后续地点通知

需要钱吗？

去公园草地，不花钱的哦～

不知道 HTML 是什么？

没关系，我用了十几年了，也不太知道这是什么。不过我会帮助你的。
